==Japanese==
PinguyBuilderをメニューから起動します。
「Settings」タブからそれぞれの設定を行います。

項目の説明
Username：EspateOSをインストール時に作成したユーザー名を入力してください。
CD Lavel：作成されたISOファイルをマウントしたりディスクに書き込んだりした際に表示されるラベル名です。32文字以内で入力してください。
FileName：ISOファイルの名前です。最後にかならず.isoとつけないとディスクイメージとして認識されません。
Working Directory：ISOを作成する際の作業ディレクトリです。ここで指定したディレクトリ内に「PinguyBuilder」ディレクトリが作成されます。通常は/homeで大丈夫です。
File to exclude：ISOファイルからあ除外するファイルを指定します。
URL for USB Creater：作成者のURLです。ディストリビューションの公式サイトや作成者のページなどを貼ると良いでしょう。
Squashfs options：squashfsを作成する際のオプションです。通常は変更する必要はありません。
Show install icon on Backup mode desktop.：バックアップモードでISOファイルを作成した際にライブ起動後のデスクトップにインストールアイコンを表示させるかの設定です。

それぞれの設定後、「Actions」タブからISOファイルを作成します。

項目の説明
Backup：バックアップモードでISOファイルを作成します。ホームディレクトリも含まれてしまうため、サイズが肥大化し、個人情報がさらされてしまう危険性があります。
Dist：ディストリビューションを作成するモードです。ホームディレクトリはないように含まれず、デフォルトのホームディレクトリは「/etc/skel」内の内容がコピーされます。
それ以外の項目はCDファイルシステムのみを構築するモードですが、動作は未確認です。

下の方にある「Select」ボタンは簡単なカスタマイズが行なえます。
上から
・ISOファイルを起動したときのGrubの背景（640x480のPNG形式のみ）
・インストール後のGrub背景
・どのユーザーの設定ファイルをデフォルトで使用するかの選択（自分で/etc/skelにコピーしたほうが確実なのでおすすめしません。）
・Plymouth Theme（起動スプラッシュテーマ）の選択ができます。デフォルトではシステムに設定されている起動スプラッシュがそのまま適用されます。
